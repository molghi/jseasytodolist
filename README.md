# Easy To-Do List - Full-Stack

A full-stack to-do list application with simple CRUD functionality.

Features include:

- Create, read, update, and delete tasks
- User authentication with signup, login, and logout
- Protected routes for authenticated users
- Persistent storage with a database (MongoDB/Mongoose)

Designed for simplicity and ease of use while demonstrating a complete full-stack workflow.

(React/Node Refresher Project)
